# Changelog

All notable changes to this project will be documented in this file.

## [0.1.1] - 2026-01-10

### Added

- `PolyBlepOsc` - band-limited oscillator using polyBLEP for reduced aliasing

## [0.1.0] - 2026-01-09

### Added

- `Oscillator` trait for building audio oscillators
- `NaiveOsc` - simple oscillator without anti-aliasing
